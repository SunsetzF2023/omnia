# 天星界

> 不知何时，古书上就这样称呼我们脚下的大地为天星界。

## 寰宇中央：四象神朝

传说在寰宇中央有四象神朝——[[../factions/四象神朝|朱雀、玄武、白虎、青龙]]。但谁也没见过有关神朝的任何子民、造物或其他记录。这是一个千古之谜。

## 修炼体系

天星界修炼 **[[../realms/武道境界|武道]]**，共十三境：武仆 → 武者(1-9星) → 武师 → 大武师 → 武宗 → 大武宗 → 武王(下中上品) → 武君 → 武圣(1-9星) → 武帝 → 武尊 → 武皇 → 武神。

- **武王**封王可劈山断水
- **武圣**成圣可增寿一万载
- **武帝**称帝可建立不朽王朝（已是传说）
- **武尊**登尊可破碎虚空
- **武神**通神可飞升上界

九洲悠悠无数载，闻名江湖的多是武圣武君，武帝已是传说，武帝之上几乎无人知晓。
