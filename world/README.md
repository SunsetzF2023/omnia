# 玄幻世界 — 世界观记录

> 用 [[双向链接]] 串联所有实体，Claude Code 可随时读取、讨论、补充。

## 目录结构

```
world/
├── README.md           ← 本文件（索引入口）
├── templates/           ← 实体模板（新建时复制）
├── realms/              ← 境界体系
├── characters/          ← 人物
├── factions/            ← 势力/宗门/王朝
├── events/              ← 世界事件/时间线
├── techniques/          ← 功法/法术/炼丹
└── locations/           ← 地图/秘境/城市
```

## 链接语法

```
[[../realms/元婴期]]     → 链接到境界
[[../characters/张三]]   → 链接到人物
[[../factions/青云宗]]   → 链接到势力
```
