# 灵族

> 非人非兽。本体多为死物，因天地造化、灵韵沾染或大能催生而诞生灵智。

## 起源

| 方式 | 说明 |
|------|------|
| **天地造化蕴养** | 在灵气浓郁之地，千年万年自然孕育灵智 |
| **沾染莫名灵韵** | 接触某种特殊的灵韵/道韵而觉醒 |
| **大能催生** | 至强者留下的痕迹蕴含其意志，日久生灵 |

## 特征

- 本体是死物——器物、山石、草木、建筑皆有可能
- 灵智诞生后拥有独立人格
- 实力取决于本体材质和蕴养时间
- 形态可幻化，但通常保留本体的某些特征

## 已知灵族

- [[../characters/匾长老|匾长老]]（天一门）— 宗门牌匾，老祖亲手题字而生灵。实力在**武圣之上**

## 与妖族的关系

- 妖族：野兽生智（血肉之躯 → 灵智）
- 灵族：死物生灵（无生命之物 → 灵智）
- 两者本质不同，但在天星界都被视为"非人"种族
