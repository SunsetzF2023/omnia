# 食灵大阵

> 胖琦与杜子腾以噬魂食灵术为媒介构建的献祭大阵。从小世界一处田地开始，逐步向外扩张。

## 基本信息

- 构建者：[[../characters/琦|胖琦]]（符修）、[[../characters/杜子腾|杜子腾]]（器修）
- 媒介：[[噬魂食灵术]]
- 初始品阶：凡品上等
- 核心机制：献祭 → 扩张 → 更多献祭

## 运作原理

### 食灵层
以大阵替代个人施法，规模吸收范围内所有尘民的**恐惧情绪**，化作阴冷灵体反馈给阵主。

### 噬魂层
吞噬范围内尘民的**神魂**，炼化后提升阵主的神魂力量。

### 献祭层
大阵威压直接**碾碎**尘民肉身，尸骸化为养料，提高大阵的强度和覆盖范围。

### 反馈循环

```
献祭尘民 → 阵域扩大 → 笼罩更多尘民 → 更多献祭 → 更强阵域
```

只要尘民足够多，理论上可以无限扩张。

## 当前状态：已崩溃

阵主神魂仅初入人品，根本维持不了大阵运转。大阵最终轰然化作虚无。

阵主 [[../characters/琦|胖琦]] 与 [[../characters/杜子腾|杜子腾]] 已离开田地，前往东方大城。

### 残余收益

- 二人神魂已从凡品升至**人品**（≈ 九星巅峰武者）
- 大阵虽崩，神魂提升的成果保留了下来

## 弱点

- 阵主（胖琦、杜子腾）本体战力一般，若阵眼被破则大阵失效
- 对大武师以上强者效果有限
