# 绝武神诀

> 段山河传遍云洲的刀修战技。武者即可入门，威力惊人。但巡天令使发现——它会将修炼者的道基潜移默化地转变为灵液。这不是武功。是域外势力对武者道途的试验。

## 基本信息

- 表面类型：刀修战技（无入门限制）
- 本质：**域外试验** — 将武者道基转化为灵液
- 创造者：[[../characters/段山河|段山河]]（受三位存在赐予）
- 门槛：极低——武者即可入门
- 传播：云洲各地皆有修习者
- 状态：⚠️ 已被 [[../factions/巡天战皇|巡天令使]] 定性——**必须销毁**

## 污染机制

巡天令使发现，绝武神诀的真正作用不是强化武者——而是**转化**。

修炼者的**道基**（武道根基）会被潜移默化地转变为某种**灵液**：
- 道基 → 灵液 → 道基被掏空
- 修炼者以为自己变强了——确实变强了。但道基正在液化
- 当道基完全转化为灵液时——修炼者会变成什么？一具空壳？还是直接死亡？

这就是天机阁爆料的"未公开的隐患"。隐患不是走火入魔——是被收割。

### 灵液：血色

转化后的灵液呈**血色**。可直接食用。

功效：
- **大幅提升对杀戮道的理解** — 灵液蕴含杀戮道韵，服用后对杀伐之道的感悟暴涨
- **补充气血** — 比任何补血丹药都直接
- **治疗心神** — 修复神魂损伤
- **提升武道资质** — 长期服用可改善根骨

这不是废物利用——道基转化的灵液是**顶级资源**。

### 丹方体系

血刀门已将灵液开发为完整的丹药体系。以不同规模的灵液凝练不同品阶的**血色丹药**：

| 灵液来源 | 丹药品阶 | 功效 |
|----------|----------|------|
| 低阶武者道基 | 凡品~人品 | 补充气血、治疗轻伤 |
| 武师~武宗道基 | 黄品~玄品 | 提升战力、加速修炼 |
| 武王以上道基 | 地品+ | 提升武道资质、突破瓶颈 |

一个云洲武者修炼绝武神诀 → 道基被转化为灵液 → 被收割 → 炼成丹药 → 喂给血刀门高层。

这就是三位存在的"建造宗门"——一座将云洲武者转化为杀戮道资源的人肉工厂。

### 杀戮道的循环

```
武者修炼绝武神诀
      │
      ▼
道基被转化为血色灵液
      │
      ▼
被收割 → 炼成丹药
      │
      ▼
血刀门高层服用 → 杀戮道感悟暴涨
      │
      ▼
更强 → 收割更多武者 → 更多灵液
```

### 为何入门极低

域外的试验不是随便挑人——是**所有人都可以成为试验品**。入门的极低门槛确保了最大范围的传播。越多人修炼绝武神诀，就有越多道基被转化为灵液。

## 段山河的角色

段山河将绝武神诀传遍云洲。三位存在给他的指令是"建造宗门"——但宗门本身或许只是载体。真正的目的可能是**大规模传播绝武神诀**，让云洲武者自愿成为灵液的生产者。

段山河本人也修炼绝武神诀。但他的道基是否也在被转化？还是他作为"选中者"可以获得豁免？

## 巡天令使的定性

[[../characters/青沧至尊|青沧至尊]] 等巡天令使已将绝武神诀定性为**污染武者道基的口诀**。

> 必须销毁。

这意味着：
- 所有修炼绝武神诀的武者——他们的道基已经被污染
- 云洲有多少武者修炼绝武神诀？不计其数
- 销毁口诀容易——但已经转化的道基怎么办？

## 特点（表面）

### 入门极易
连 1 星武者都能掌握基础。这不是优点——是陷阱。

### 上限未知
修炼者境界越高，转化的灵液品质越好。这解释了为什么绝武神诀"上限未知"——因为域外试验需要各个境界的数据。

### 与血刀的配合
绝武神诀与 [[../factions/血刀门]] 的血脉炼刀法门相辅相成。以血气驱动的绝武神诀威力倍增——转化的速度也倍增。

## 流传影响

云洲各地武者皆可修习。低阶武者战力因此整体提升——但代价是什么？

段山河公开绝武神诀时，云洲武者视其为慷慨。现在知道——那是在撒种。
