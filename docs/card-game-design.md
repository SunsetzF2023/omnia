# 勇者牌局 — 设计文档（WIP）

> 月圆之夜风格的单人卡牌 Roguelike，战士职业首发。
> 状态：数据调研中，代码尚未开始。

---

## 一、核心循环

```
开局取名 → 森林妖精三选一天赋 → 进入第1章
  → 分支选路 → 战斗/商店/宝箱/绷带
  → 战斗胜利选牌 → 升级三选一 → Boss
  → 第2章 → 第3章 Boss
  → 通关/阵亡 → 评分 → 排行榜
```

## 二、战斗引擎

### 回合流程
```
回合开始 → 抽满手牌上限(默认5张)
  → 玩家出牌(消耗AP，默认3点/回合)
  → 结束回合
  → 怪物行动(意图系统，提前显示)
  → 状态结算(中毒/灼烧/虚弱/易伤)
  → 回到回合开始
```

### 核心规则
| 机制 | 规则 |
|------|------|
| 护盾 | ✅ 跨回合累积，不清零 |
| 抽牌 | 每回合补至手牌上限 |
| 牌堆 | 打光后弃牌堆洗入牌堆 |
| AP | 每回合回满 3 点 |
| 意图 | 敌方下回合动作可见（攻击/防御/蓄力） |

### 状态效果
| 状态 | 效果 |
|------|------|
| 虚弱 | 攻击力 -25% |
| 易伤 | 受伤 +50% |
| 力量 | 攻击伤害 +N |
| 中毒 | 每回合扣 N HP |
| 灼烧 | 每回合扣 N HP（比中毒高） |

## 三、升级系统

- 击杀怪物获得经验值
- 升级时：最大HP提升 + 三选一
  - 🃏 选一张牌加入牌组
  - ✋ 手牌上限 +1
  - ⚔️ 装备槽 +1（初始 1 个，可扩展）

## 四、主动技能（战士，2个）

| 技能 | 效果 | 冷却 |
|------|------|------|
| 🪞 镜像 | 复制下一张打出的牌 | 8 场战斗 |
| 🛡️ 坚毅 | 获得等于已损失HP的护盾 | 8 场战斗 |

每场战斗只能用一次主动技能。

## 五、卡牌类型

| 类型 | 消耗 | 说明 |
|------|------|------|
| ⚔️ 攻击牌 | AP | 造成伤害 |
| 🛡️ 防御牌 | AP | 获得护盾 |
| ✨ 技能牌 | AP | BUFF/DEBUFF/抽牌等 |
| 🎒 装备牌 | AP | 装备后获得被动效果 |

## 六、战士初始牌组（10张）

| 牌名 | 费用 | 类型 | 效果 |
|------|------|------|------|
| 斩击 ×5 | 1⚡ | 攻击 | 造成 6 伤害 |
| 格挡 ×3 | 1⚡ | 防御 | 获得 5 护盾 |
| 重击 ×2 | 2⚡ | 攻击 | 造成 10 伤害 |

## 七、战士初始装备

每回合开始时获得 +2 护盾。装备槽初始 1，可升级到 2。

## 八、地图系统

- 3 章，每章 5 层节点（含 Boss）
- 节点类型：⚔️战斗 / 👹精英 / 🛒商店 / 🏕️休息 / ❓事件 / 📦宝箱 / 🩹绷带 / 💀Boss
- 每层 2-3 条路径可选

## 十、商店
- 买牌（随机 3 张）
- 删牌
- 买装备

## 十一、宝箱 / 绷带
- 📦 宝箱：随机给 1 张牌（好或烂）
- 🩹 绷带：3/3，每次恢复最大HP的 3%

## 十四、评分系统
```
评分 = 基础分(消灭数×10 + Boss×50) × 章节倍率
      + 装备分
      + 金币分
      + 通关加成(+200)

评级：F(<100) D(100) C(200) B(350) A(500) S(700) SS(1000)
```
